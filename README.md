# Nebula Music Player 🎵

## Overview

**Nebula Music Player** is a modern and visually interactive web-based music player designed with a sleek UI and engaging animations. It provides a personalized listening experience by greeting users and allowing them to interact with a stylish music player interface.

The project focuses on **clean UI design, animated elements, and smooth user interaction**, making it suitable for portfolio projects or front-end showcases.

---

## Features ✨

* 🎧 **Modern Music Player Interface**
* 🎵 **Animated 3D Music Loader**
* 👋 **Personalized Welcome Modal**
* 🎨 **Glassmorphism UI Design**
* 📱 **Responsive Layout**
* 🎼 **Floating Music Notes Animation**
* 🔊 **Equalizer Animation**
* 👤 **User Greeting System**

## How to Run the Project 🚀

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your browser.

No installation or server setup is required.

---

## How to Add Songs 🎶

To make songs work in the player:

1. Create a **songs** folder in the project directory.
2. Add your **.mp3 files** inside it.
3. Connect the songs inside **script.js**.

Example:

```javascript
const songs = [
  {
    name: "song1.mp3",
    title: "Song Title",
    artist: "Artist Name"
  }
];
```
